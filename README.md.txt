# 🎉 Gestor de Convidados (RSVP)

Aplicação web para gestão de convidados com confirmação de presença em tempo real.

---

## 🚀 Funcionalidades

- Adicionar convidados
- Confirmar presença
- Contador automático
- Interface simples e funcional

---

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript

---

## 💡 Objetivo

Praticar manipulação do DOM, lógica de estado e criação de aplicações interativas.
